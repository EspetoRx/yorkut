## Sobre o Yorkut

Clone do Orkut feito em Laravel + Vue.JS 3.
Aplicação Dockerizada

## Reproduza

Faça o clone e rode

``cp .env.example .env``

``docker compose up --build -d``

``docker exec yorkut_app php artisan migrate``
